## 🎯 产品定位

### 一句话介绍
**YUIChat** - 企业级 AI 知识库管理平台，基于 RAG 技术实现智能文档问答。

### 产品愿景
让企业知识触手可及，通过 AI 智能问答降低知识获取成本，提升工作效率。

## ✨ 核心功能清单

### 1️⃣ 知识库管理

| 功能 | 说明 |
|------|------|
| **多项目管理** | 创建多个独立知识库项目，数据完全隔离 |
| **文档上传** | 支持 PDF、Word、Excel、PPT、TXT 等多格式 |
| **网页爬取** | 输入 URL 自动抓取网页内容，支持 JavaScript 渲染 |
| **文档状态跟踪** | 实时显示文档处理状态（处理中/已完成/失败）|
| **文档重新学习** | 支持重新处理已上传的文档 |
| **批量操作** | 支持批量删除文档 |

### 2️⃣ 智能对话

| 功能 | 说明 |
|------|------|
| **RAG 问答** | 基于上传文档的智能问答，回答有据可依 |
| **流式输出** | 实时流式显示 AI 回复，体验流畅 |
| **多轮对话** | 支持上下文连续对话，理解对话历史 |
| **引用来源展示** | 显示回答的来源文档片段，支持溯源 |
| **性能模式选择** | 快速模式 (GPT-4o-mini) / 精准模式 (GPT-4o) |
| **语义缓存** | 相似问题秒级响应 (<500ms)，降低 API 成本 |
| **对话记录** | 自动保存对话历史，支持历史回溯 |

### 3️⃣ 推荐问题系统

| 功能 | 说明 |
|------|------|
| **智能推荐问题** | 基于文档内容自动生成推荐问题 |
| **多语言支持** | 同时生成中/英/日三语言问题 |
| **Follow-up 问题** | 对话后推荐相关后续问题 |
| **向量检索优化** | 使用语义相似度筛选高质量问题 |
| **自定义推荐问题** | 支持手动配置推荐问题 |

### 4️⃣ 项目设置

| 功能 | 说明 |
|------|------|
| **项目信息管理** | 修改项目名称、描述 |
| **AI 头像设置** | 自定义 AI 助手头像 |
| **欢迎语配置** | 多语言欢迎语设置 |
| **推荐问题配置** | 手动设置推荐问题 |
| **性能模式设置** | 选择快速/精准模式 |
| **自动翻译填充** | 一键将欢迎语/问题翻译到其他语言 |

### 5️⃣ 外部分享

| 功能 | 说明 |
|------|------|
| **项目级分享** | 一个链接分享整个项目的所有文档 |
| **免登录访问** | 外部用户无需登录即可使用 |
| **分享链接管理** | 支持重置分享链接 |
| **多语言界面** | 分享页面支持语言切换 |

### 6️⃣ 数据统计

| 功能 | 说明 |
|------|------|
| **对话统计** | 总对话数、今日对话、总消息数 |
| **来源区分** | 区分测试对话和公开用户对话 |
| **高频问题排行** | Top 10 高频问题统计 |
| **对话记录管理** | 查看、筛选、搜索对话记录 |
| **数据保存周期** | 设置数据保存时长 (永久/3月/6月/1年) |

## 🔧 文档处理流程

### 完整处理流程图

```
┌───────────────┐     ┌───────────────┐     ┌───────────────┐
│  1. 文件上传   │ ──▶ │  2. 文档解析   │ ──▶ │  3. 文本切分   │
│  Supabase     │     │  PDF/Word/    │     │  1000字/块    │
│  Storage      │     │  Excel/PPT    │     │  200字重叠    │
└───────────────┘     └───────────────┘     └───────────────┘
                                                    │
                                                    ▼
┌───────────────┐     ┌───────────────┐     ┌───────────────┐
│  6. 智能问答   │ ◀── │  5. 向量检索   │ ◀── │  4. 向量存储   │
│  GPT-4o 生成  │     │  相似度搜索   │     │  pgvector     │
│  回答         │     │  Top-K 结果   │     │  Embedding    │
└───────────────┘     └───────────────┘     └───────────────┘
```

### 支持的文件格式

| 格式 | 说明 |
|------|------|
| **PDF** | 使用 PyPDF 解析，支持文本提取 |
| **Word (.docx)** | 使用 Docx2txt 解析 |
| **Excel (.xlsx/.xls)** | 使用 UnstructuredExcelLoader 解析 |
| **PowerPoint (.pptx)** | 使用 python-pptx 解析 |
| **文本文件 (.txt)** | 自动编码检测 |
| **网页 URL** | Selenium + Chrome 爬取，支持 JS 渲染 |

## 🚀 性能优化

### 问答语义缓存

| 指标 | 缓存未命中 | 缓存命中 |
|------|-----------|---------|
| 响应时间 | 3-7秒 | <500ms |
| API 成本 | 标准费用 | ¥0 |
| 准确性 | 实时生成 | 与首次相同 |

**特性**：
- 相似度阈值 0.95，高精度匹配
- 缓存有效期 7 天
- 热门问题自动延长至 28 天
- 知识库更新时自动清除缓存

### 性能模式对比

| 模式 | 使用模型 | 响应时间 | 适用场景 |
|------|---------|---------|---------|
| ⚡ 快速模式 | GPT-4o-mini | ~1秒 | 实体机器人、电话客服 |
| 🎯 精准模式 | GPT-4o | ~10秒 | 文字客服、深度分析 |
